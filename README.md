# Casino Edge Companion

**▶ Use it live: https://sleeprived.github.io/casino-edge-companion/**

A phone-friendly, offline web app that teaches the *mathematically correct* play
for casino table games and shows the real odds — so you play smarter and lose
slower. Open the link on any device, or **Add to Home Screen** to install it.

## Features
- **Blackjack Trainer** — drills perfect basic strategy and scores you.
- **Video Poker** — finds the optimal cards to hold and the exact expected value.
- **Craps** — every bet's true odds and house edge, plus a sucker-bet quiz.
- **Roulette** — American vs European edges for every bet.
- **Baccarat** — the three bets and why you always bet Banker.
- **House-Edge Cheat Sheet** — every bet in the app ranked best-to-worst.

## Note
A learning tool, not a betting system — every game still favors the casino; this
just helps you make the smartest available play. Nothing leaves your device.
